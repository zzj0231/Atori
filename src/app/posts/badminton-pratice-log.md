从去年 10 月开始接触羽毛球，一直都是打球娱乐的状态。最近发现技术进步有点慢，想要有针对性地训练一下，所以开始记录这个训练过程。

## 训练目标

**时间段**：2025年8月-10月

**重点技术**：后场球技术

- 杀球
- 高远球
- 收吊球
- 后场步伐

**技术分类**：

- 前场球：网前推球、搓球、勾球等
- 后场球：杀球、收吊球、高远球、滑板等
- 步伐：前场步伐、后场步伐、中场步伐
- 发球：高远球发球、平快球发球等

| -------- | -------- | ----------------------------------------------------------------------------- | -------- | ---- |

| 后场球   | 杀球     | 原地杀球10分钟＋上步杀球10分钟                                                | ⭕       | 8/12 |
| 发球     | 发球     | 发球 6×10 个；要点：挥拍一致性、腿—腰—臂传导发力                              | ⭕       | 8/13 |

**第1周总结**：
杀球：

- 引拍时球柄要松，内旋打击时刻就是击球点，决定了球的向下or飘；
- 转身时，必须腰弓才能给左腿移动的空间
- 获得反弓的力量，首先需要保证转体时，右边必须压得比较紧
- 左脚转完引拍结束，落地时完成内旋发力，这个时候如何通过左脚将力量借给内旋？
- 步伐连贯的核心在于回动结束与启动开始时重心调整的重合
- 上网步伐: 启动步 -> 右脚蹬地 -> 左脚跨步 -> 跟步回位
- 后撤步伐：启动步 -> 左脚后交叉 -> 右脚落地缓冲 -> 并步回中
- **实践问题：手紧、核心松、转体做不出来、回动慢**
  - 步伐调整: 并步+跨步代替连续脚尖跳；用大腿和臀部驱动; 早启动 + 预判
  - 杀球调整: 转身时引拍过程手要松，否则引拍做不出来；身子绷紧+靠身体传导力量
  - 引拍时想象用手肘去碰身后的墙，迫使肩关节打开
  - 转身时想象髋部撞开一扇旋转门，启动核心爆发力

### 第2周 (8月16日-8月20日)

- 重点: 腹肌训练(15分钟) + 启动训练(开合/转髋)(1分钟 ) + 步伐(并步) + 转体引拍(10分钟)

| 技术类型 | 技术名称  | 训练内容                                                                             | 是否完成 | 日期 |
| -------- | --------- | ------------------------------------------------------------------------------------ | -------- | ---- |
| 后场球   | 启动+杀球 | 原地开合跳快速坚持30s，脚后跟不落地(3组)；髋部转动带动双脚，慢20s快20s，脚后跟不落地 | ❌       | 8/16 |
| 后场球   | 启动+杀球 | 原地开合跳快速坚持30s，脚后跟不落地(2组)；转体引拍(30次)                             | ⭕       | 8/17 |
| 后场球   | 启动+杀球 | 转体引拍(20次)                                                                       | ⭕       | 8/18 |
| 后场球   | 启动+杀球 | 扔球击球(20次)                                                                       | ⭕       | 8/19 |
| 综合     | 杀球      | 扔球击球(40次)；原地开合跳快速坚持30s，脚后跟不落地(2组)                             | ⭕       | 8/20 |

**第2周总结**：

- 启动： 对方触球瞬间双脚离地，给予身体一个球落点方向上的推力
- 启动类型：开合启动、开合交叉启动(双脚收回时一前一后)、转髋启动(转髋瞬间完成启动)、前后交叉启动、小跳启动
- 击球后先调整重心，收回到双腿中间
- 并步核心：启动时压低重心，直到重心转换时再蹬起
- 马来步核心： 启动时，右腿转左腿，蹬起时转换重心
- 步伐的核心不是看跑的有多快，而是击球后快速收回重心并根据自己的出球选择合适的位置准备
- 重心调整速度和球速成正比，先把重心收回到左脚(更靠近取位的脚，取位是下次蹬地的位置)
- 下压：大臂不动，转体传导力量到前臂，前臂再让手腕下压，力传导只有大臂不动，才能把力量传导出去
- 腹肌训练：反向飞鸟弯举(30秒)、跨步登山(30s,身体不发生明显侧转)、负重卷腹、前平举锤式弯举(30s)、波比跳(两次俯卧撑，跳起来)、俄罗斯转体(30s，腰稍弯腹肌发力，负重在身前) 一组6个动作、间隔15s，一周期5分钟；3组15分钟
- **实践问题：击球后，重心调整不到位，手紧球发力感弱**

| 综合     | 杀球      | 扔球击球(20次)；                                 | ⭕       | 8/28 |

**第3周总结**：

- 较第一周，身体后退可自然转体
- 较第一周，转体后，腰弓不明显
- 较第二周，经常性击球后，重心未调整回双腿中间；
- 较第二周，可通过转体，压腕增加球下落力量，但未到达鞭打感
- 步法不到位导致身体在击球时处于失衡状态，只能靠手臂和手腕勉强处理
- 牢记“用脚找球，转身发力“。忘记“用手打球”。
- 击球有力，需要好的击球点。好的击球点，需要步伐到位
- 保持击球点在你右眼的前方(20~30cm), 高球和杀球需要保持击球点在你右眼正上方
- 准备动作：右脚稍后（右手持拍），脚后跟抬起，身体半侧向球网，双手举起（非持拍手保持平衡，持拍手手肘抬高，拍头置于脑后）

- **实践问题：力量传导不到位，鞭打感差，手紧球发力感弱**
- 普通击球时，击球空间小，只能通过推手腕将球打出，球速较慢、球轨道较平，容易接。
- 击球点落在身体后方或侧面过于贴近身体的位置，身体力量（腿、腰腹）完全用不上，只能孤立地依靠手臂和前推手腕来“捅”球。
- 没有提前做好侧身和引拍准备。当球来时，才从正对球网的姿势临时转身抬手，错过了最佳的发力时机。

**第5周总结**

- 引拍时，先压住右边，放左边的同时，因反弓向上
- 不是顶肘，而是旋肘；旋肘时保持小臂与肘垂直
- 更有力量: 反弓越多、转换重心速度越快(左脚放的同时，右脚跳起来)
- 力量传递的好，手指会自然抓紧
- 力传导的节点：腿、腰、大臂、小臂、手腕

- 良好的杀球，球拍头要能感受到击球的重量，同时会明显感受到，球头给予球拍一个反作用力
